# 2. Use next.js and typescript

Date: 2026-06-23

## Status

Accepted

## Context

We need to have some framework to build the application on,
with both API and UX support.

## Decision

We are going to use next.js with typescript.

## Consequences

I have used typescript before, and Node.js, but not Next.js.
This introduces some risk, especially given the 2 hour time
limit for an initial working version, but is a good exercise
for working with it in the future, and is optimal for the
interviewing team to evaluate.
